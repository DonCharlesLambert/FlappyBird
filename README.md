## Get it Running
1. Open Terminal  

2. Clone the Repo  
`C:\> git clone https://github.com/DonCharlesLambert/FlappyBird`  

3. Enter the Repo  
`C:\> cd FlappyBird`  

4. Run the Game  
`C:\FlappyBird> java Main.java`
